# Sealed — タイムカプセルSNS（プロトタイプ）

**仮称:** Sealed / またね / あとびらき

投函は今、開封は未来。身内だけのタイムカプセルSNS。

## 開き方

```bash
open /Users/yushinnuuuu/sealed/index.html
# または
cd /Users/yushinnuuuu/sealed && python3 -m http.server 8766
# → http://127.0.0.1:8766
```

## コンセプト

| 機能 | 内容 |
|------|------|
| **封じる** | 言葉・写真・音声を投函。その場では誰も中身を見られない |
| **薄い可視性** | 「今日4人が封じた」だけ分かる |
| **開封タイミング** | 3日 / 7日 / 1ヶ月 / 半年 / 1年 |
| **あとで見る** | 期間別の件数を可視化（例: 3日後 3、1年後 25） |
| **開封式** | 全員揃った瞬間、またはアプリを開いた日に一斉開封 |
| **お題モード** | Fortune Spark 風のお題に沿って投函 |
| **宛先** | グループ宛 / 未来の自分宛 |

状態は `localStorage`（`sealed_v1`）。
